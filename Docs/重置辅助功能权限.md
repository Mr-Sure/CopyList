# 重置 CopyList 辅助功能权限

当自动粘贴功能失效或提示「不允许发送按键」时，可按以下步骤重置权限。

## 方法一：系统设置中移除并重新添加

1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 点击左下角 🔒 解锁
3. 查找以下应用并**移除**（点击 `-` 号）：
   - **CopyList**（如果存在）
   - **Terminal** 或 **终端**（如果存在）
4. 重新打开 CopyList，系统会再次弹出授权请求
5. 点击「好」授予权限

## 方法二：重置整个 TCC 数据库（慎用）

> ⚠️ 此操作会重置所有应用的辅助功能权限，需要重启电脑后生效

```bash
tccutil reset Accessibility
```

## 方法三：检查当前权限状态

在系统设置的辅助功能列表中查看：

| 状态 | 含义 | 操作 |
|------|------|------|
| CopyList ✅ 已勾选 | 权限正常 | 如仍失效，重启应用 |
| CopyList ❌ 未勾选 | 权限被关闭 | 勾选即可 |
| 列表中无 CopyList | 未授权 | 点击 `+` 添加 |

## 验证

权限设置完成后：

1. 重启 CopyList 应用
2. 点击状态栏图标，选择一条历史记录
3. 确认内容自动粘贴到当前应用
