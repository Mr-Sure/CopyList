# CopyList 自动粘贴权限设置

CopyList 的自动粘贴功能通过模拟 `Cmd+V` 实现，需要 macOS **辅助功能**权限。

## 授权方法

### 方法一：首次启动自动引导

首次启动 CopyList 时，如果未检测到辅助功能权限，会自动弹出引导窗口，点击「打开设置」即可跳转到系统设置页面。

### 方法二：手动设置

1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 点击左下角 🔒 解锁
3. 找到 **CopyList** 并勾选（如列表中不存在，点击 **+** 号手动添加）

### 方法三：命令行触发

在终端运行以下命令，系统会自动弹出权限请求：

```bash
osascript -e 'tell application "System Events" to keystroke "test"'
```

点击「好」允许权限。

## 重要提示

- **本应用已经过代码签名**
- **首次授权后，重新安装不需要再次授权**
- Bundle ID 固定为 `com.local.copylist`
- 如果更新后权限失效，请参考 [重置辅助功能权限](重置辅助功能权限.md)

## 验证

设置完成后，在微信等应用中测试：

1. 点击输入框获取焦点
2. 点击 CopyList 状态栏图标
3. 点击一条历史记录
4. 内容应自动粘贴到输入框

## 不想开启自动粘贴？

在设置中关闭「自动粘贴」开关，应用仍可正常使用：

- 点击历史项 → 复制到剪贴板
- 手动 `Cmd+V` 粘贴
