# CopyList 自动粘贴权限设置

## 问题
CopyList 的自动粘贴功能需要"辅助功能"权限才能工作。

## 解决方法

### 方法一：通过系统设置
1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 点击左下角 🔒 解锁
3. 找到并勾选以下应用（可能需要手动添加 + 号）：
   - **Terminal** 或 **终端**
   - **osascript**
   
### 方法二：测试触发权限请求
在终端运行以下命令，系统会自动弹出权限请求：

```bash
osascript -e 'tell application "System Events" to keystroke "test"'
```

点击"好"允许权限。

## 验证
设置完成后，在微信等应用中测试：
1. 点击输入框
2. 点击 CopyList 菜单栏图标
3. 点击历史记录
4. 内容应该自动粘贴到输入框

---

如果还是不行，请尝试重启应用或系统。
